서비스 소개

보험 계약 정보를 기반으로 대출 가능 금액을 자동 산출하고, 신청·심사·정산까지 이어지는 금융 프로세스를 자동화한 핀테크 서비스입니다. Codef API를 통해 보험 계약 정보를 조회하고, 유동화 분석 엔진으로 대출 가능 금액을 산출하는 전체 백엔드 시스템을 단독으로 설계·구축했습니다.

주요 기능

  • Codef 기반 보험 조회 — Connected ID 생성, 보험사별 계약 정보 조회, 저축성·보장성 구분, Transaction 로그
  • 유동화 분석 엔진 — 해약환급금·계약대출가능액·보험담보대출가능금액 기반 대출 가능 금액 자동 산출
  • 대출 신청·심사 — 대출 신청(필요 금액·사용 기간), 상태 관리(대기·처리완료·거절), 유동화 종류별 Item 관리
  • 본인인증 — 주민등록번호·통신사·휴대폰 기반 본인인증, Codef Connected ID 연동
  • 백오피스 — Django Admin 기반 대출 심사, 파라미터 관리, 사용자 관리, 보험 거래 내역 조회

핵심 개발 내용

  • Codef API 통합 — 보험사 계약 정보 조회, Connected ID 생성 및 관리, Transaction 로그 추적
  • 유동화 분석 엔진 — 해약환급금·계약대출·보험담보대출 기반 대출 가능 금액 및 이율 자동 산출
  • 대출 심사 워크플로 — 신청·심사·승인 상태 관리, 유동화 종류별 금액·이율 관리
  • PostgreSQL Aurora Master-Slave — 읽기/쓰기 분리로 조회 성능 최적화, 트랜잭션 일관성 보장
  • Celery 비동기 처리 — 보험 조회·정산·알림 백그라운드 처리, 재시도 로직
  • Django Admin 백오피스 — 대출 심사 관리, 보험 거래 내역 조회, 파라미터 설정
  • AWS ECS Fargate — 컨테이너 기반 배포, RDS Aurora, S3 스토리지

  • 유동화 분석 알고리즘 — 저축성·보장성 보험별 대출 가능 금액 산출, 계약대출·중도인출·보험담보대출 조건별 이율 계산
